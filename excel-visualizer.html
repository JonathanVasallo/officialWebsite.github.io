Summary
Have you ever looked at a spreadsheet and thought 'what does any of this mean?'.
Well look no further, the Excel Visualizer is the solution you didn't know you were looking for.
This piece of software allows any 'average joe' to upload a excel file (.csv) of two formats into the program.
The formats are dependent on this information inside of the spreadsheet. One type of format is to have a total of 2 columns and any number of columns above 0. The Second option is to have the total number of Rows and Columns both under 10 individually, and to have the same amount of rows and columns. Once the conditions have been met the software will realize which of the two conditions you have given and store all of your information.
To release the information the program awaits your selection in the navigation bar located in the bottom of the Graphical User Interface (GUI).
Functionality
Once you have successfully added the excel file (.csv) there will be 6 other buttons to perform actions throughout the program.
The first two buttons you'll see; 'Show Bar Chart' and 'Show Pie Chart' are interchangeable as pressing one will display that type of chart from the data you provide and make the other chart disappear.
The next button 'Customize Axis / Title' allows you to name the title and axis of the currently chosen graph you are viewing. There are no restrictions to the titles you choose.
If you need to filter out some data that is either out of the mean / median, the 'Bar Chart Ranges' button allows you to have a set number that should not be exceeded when displaying the bar chart.
    - X and Y values are determined by a scrollable meter.
     - Values chosen are the maximum values that will be displayed in the Bar Chart alone.
All done examining you data? What better way to keep that image in your head than just to save it as a picture file (.PNG) and access it just about anywhere!
- Find the 'Save Graph' button
- Follow the instructions
- Name the file and store it accordingly
Finished with the first excel sheet? You can either...
- Click the 'Read in Csv File' to load in another excel file and repeat
- Click the 'Quit Program' to successfully close the software.
Goal
Allow any user to quickly analyze data without the hassle of graphing each and every data point.
- Save time
- Increase productivity
