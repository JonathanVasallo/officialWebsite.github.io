Create Family Trees

Download on GitHub

Summary
Organizing family structures is a hassle from knowing who is related to who, to which generation is a family member is from.
Stress no longer, the tree generator is here to save you the headache.
This software takes in a text file (.txt) in as input in order to build corresponding relationships amongst the family provided.
Lets take me and my family for example, and lets also assume that you have a sister, a mother and a father.
On your end you will type out each family member including yourself, their birthday, and location (optional)
- Father, 3 19 1967, Killington, VT
- Mother, 5 30 1972, Killington, VT
- Sister, 1 19 1994, Killington, VT
- Me(you), 1 6 1998, Killington, VT
Then simply type your fathers name for example and state his relationship to the rest of your family.
- Father, parentof, Jonathan Vasallo(Me)
- Father, parentof, Sister
- Father, marriedto, Mother
Save this text file for loading. Once you start the program create a profile and add your families text file.
All work has been completed for the proper use of the program, now we can select the 'Draw Family Tree' button to display all relationships.
Functionality
The 2-D Canvas displaying the Family tree is fully interactive. The plane upon which the names are lying are responsive to clicks.
Clicking on a persons name on the displayed tree triggers and info box pop-up for the user to view.
The info box contains...
- Name
- Date of Birth (if added to the text file originally)
- Spouse
- Children
- Parents
- Grandparents (if added)
- Grandchildren
- Nieces / Nephews
- Aunt / Uncles
If you decide that there is an error with the information you can remove that specific person from the tree and simply add that same family member back in a separate text file with their correct info OR reupload the edited text file.
Goal
- Save time
- Provide a useful tool
